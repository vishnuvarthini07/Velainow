About Velainow

Velainow is a hyperlocal job matching platform that intelligently connects clients to nearby skilled workers based on location and skills. Think Rapido/Ola – but for daily-wage or skill-based work like plumbing, electrical, delivery help, etc.

Key Features

  • Real-time task posting and worker registration
  • Auto-assignment of workers based on skill and pincode
  • Status update of worker (Available/Busy)
  • Built using Firebase (Realtime Database), Tailwind CSS, and Vanilla JS

Why this matters?

We aim to solve the everyday need of finding verified nearby help quickly, using simple but powerful tech in a student-built prototype for CodeCubicle 4.0.
